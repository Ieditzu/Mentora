MENTORA – PREZENTARE SLIDEV

Pornire în modul de prezentare:

```bash
cd presentation-slidev
npm install
npm run dev
```

Build static:

```bash
npm run build
```

Export PDF:

```bash
npm run export
```

Apăsați `P` în timpul prezentării pentru Presenter Mode. Notițele fiecărui slide indică persoana care vorbește și ideile principale. Imaginile folosite sunt capturi reale din directorul `images/` al proiectului, copiate în `public/` pentru buildul Slidev.
